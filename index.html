Welcome to Documents!

This feature of jihadology.net is designed to be a resource for journalists, columnists, academics and anyone else researching or writing about Middle Eastern terrorist groups.

The primary benefit of docs.jihadology.net is its usage of open-source code from The New York Times called Emphasis which allows direct linking to specific paragraphs and sentence highlighting.

Using Emphasis

  1. Double tap the shift key and ¶ symbols appear beside paragraphs
  2. Click any ¶ to select the paragraph
  3. (Optional) Click one or more sentences in a selected paragraph to highlight it
  4. Copy/paste the generated URL in your address bar to link directly to the selected paragraph with the desired highlighting

Learn more about this project by visiting us on Github.
